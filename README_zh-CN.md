# 关于 facecto-code-base-utils
facecto-code-base-utils 是一套工具集，它将聚合更多的工具。

# 快速开始
## 步骤 1: 配置 pom.xml 依赖
```
<dependency>
  <groupId>com.facecto.code</groupId>
  <artifactId>facecto-code-base-utils</artifactId>
  <version>1.0</version>
</dependency>
```
